ReferenceRecu — Widget "Reçus par mois"

Ce widget affiche :
- Une liste déroulante (NomProjetString)
- Un graphique (nb de Recu par mois)
- Clic sur une barre => sélectionne les lignes de ce mois dans Grist

Dans le panneau de droite du widget:
- NomProjetString
- Recu
- RecuString
- Archive

Notes:
- Le widget ignore Archive=true
- Le widget ignore les dates 1900-01-01
